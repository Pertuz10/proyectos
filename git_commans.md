Describe que hacen los siguientes comandos de git, escribe su descripcion al frente en una sola linea.

git status: Este comando se usa para obtener el estado actual del repositorio

git clone:Este comando se utiliza para clonar un repositorio de Git, creando una copia local del repositorio remoto. 

git pull:Este comando se utiliza para fusionar los cambios del repositorio remoto en el repositorio local. 

git checkout: Este comando se utiliza para cambiar de rama o restaurar archivos en el directorio de trabajo

git log:Este comando muestra el historial de confirmaciones (commits) en el repositorio. 
 

git branch:Este comando muestra las ramas existentes en el repositorio y permite crear, eliminar o cambiar entre ramas. 
 

git add: Este comando se utiliza para agregar cambios al área de preparación (staging area) para ser confirmados

git commit:Este comando se utiliza para confirmar los cambios realizados en el repositorio local. 

git push:Este comando se utiliza para enviar los cambios confirmados al repositorio remoto. 

git merge:Este comando se utiliza para fusionar los cambios de una rama en otra rama del repositorio.
